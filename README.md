# Padavan-build说明

直接Fork到自己的仓库，修改好 .github/workflows/build-padavan.yml 文件之后，点击右上角的 Star 星星按钮即可开始自动编译。

(重新编译点击 Unstar 星星按钮后再点击 Star 星星按钮即可重新编译)
